Add elegant pagination themes to WP-Pagenavi.  WP Visual Pagination provides unlimited customization and an easy to use interface to WP-Pagenavi.

== Description ==

WP Visual Pagination is an advanced pagination theme framework built on top of the popular [WP PageNavi](http://wordpress.org/extend/plugins/wp-pagenavi/) plugin with pre-loaded professionally designed pagination themes. WP Visual pagination allows you create or customize pagination themes for an unlimited number of variations to exactly match the look and feel of your blog. If you need more customization take a look at [Visual Pagination Pro](http://www.wppluginpros.com/shop/wp-visual-pagination-pro/).

<h3>FEATURES</h3>

<h3>Create Stunning Pagination Themes</h3>
* Complete control over many aspects of your pagination theme settings
* Supports CSS based pagination theme designs with unlimited color picking combination
* Preview  theme changes  from back-end before you save
* Powerful pagination theme framework allows for complex pagination designs
* No HTML Knowledge required – easy to use interface
* Simply pick colors to match your current theme and save!

<h3> Customizable Professional Themes </h3>
* Now includes 7 Premium customizable themes
* Unlimited customization of colors
* For more customization check out [Visual Pagination Pro](http://www.wppluginpros.com/shop/wp-visual-pagination-pro/) our top of the line fully customizable pro version.

<h3> Easily Integrate in Your WordPress blog or Website </h3>

== Installation ==

= If you already have WP-Pagenavi installed: =
* Upload the WP Visual Pagination plugin to your WordPress
* Activate plugin in plugins menu

= If you DO NOT already have WP-Pagenavi installed: =
* Upload the WP Visual Pagination plugin to your WordPress
* Activate plugin in plugins menu
* Follow-up Setup for [here](http://wordpress.org/extend/plugins/wp-pagenavi/installation/).  There is also quite a bit of help for installations for specific themes [here](http://wordpress.org/support/plugin/wp-pagenavi).
